Instagram

Posts que aparecen en la sección "Lo que pasa en el fuego" del sitio

Configuración inicial requerida
Ejecutá este SQL en Supabase → SQL Editor:
CREATE TABLE IF NOT EXISTS instagram_posts ( id SERIAL PRIMARY KEY, url TEXT NOT NULL, activo BOOLEAN DEFAULT true, orden INT DEFAULT 0, created_at TIMESTAMPTZ DEFAULT now() ); ALTER TABLE instagram_posts DISABLE ROW LEVEL SECURITY;
💡 Para obtener la URL de un post: andá al post en Instagram → clic en los tres puntos → Copiar enlace. Pegá el link acá.
Formato: https://www.instagram.com/p/ABC123/

Imágenes del Sitio

Cambiá las fotos fijas del sitio sin hacer deploy. Los cambios se aplican al instante.

Configuración inicial requerida
Ejecutá este SQL en Supabase → SQL Editor, una sola vez:
CREATE TABLE IF NOT EXISTS site_settings ( key TEXT PRIMARY KEY, value TEXT NOT NULL, updated_at TIMESTAMPTZ DEFAULT now() ); ALTER TABLE site_settings DISABLE ROW LEVEL SECURITY; INSERT INTO site_settings (key, value) VALUES ('hero_image', 'img/p03.jpg'), ('about_image', 'img/p01.jpg') ON CONFLICT (key) DO NOTHING;
